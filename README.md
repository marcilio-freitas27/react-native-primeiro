### Instalação 

- Clonar projeto

```

$ git clone https://github.com/<nome do repositorio>
```

- Frontend

```
$ cd <nome do repositorio>/primeiro

$ npm i 

$ npm start

Opção w - web

```


- Backend

```
$ cd <nome do repositorio>/backend-api-sql

$ npm i 

$ nodemon server.js

```
